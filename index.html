Skills & Technologies

These are the technologies and tools I use to build modern, scalable, and performance-optimized web applications. I focus on writing clean, reusable, and maintainable code.

HTML5

HTML5 is the foundation of all my web projects. I use semantic HTML5 elements to create well-structured, SEO-friendly, and accessible web pages. This ensures better performance, improved search engine visibility, and compatibility across all modern browsers.

CSS3

CSS3 allows me to design visually appealing and responsive layouts. I work with Flexbox, CSS Grid, media queries, transitions, and animations to build modern user interfaces that adapt perfectly to mobile, tablet, and desktop screens.

JavaScript (Core & Advanced)

JavaScript enables me to add interactivity and dynamic behavior to websites. I work with DOM manipulation, event handling, asynchronous programming (Promises and async/await), API integration, and browser storage to build real-world functionalities.

Bootstrap

Bootstrap is a responsive CSS framework that helps me rapidly build mobile-first layouts. I use Bootstrap components such as grids, navigation bars, modals, and forms to maintain consistency and speed up development.

Tailwind CSS

Tailwind CSS is a utility-first framework that gives me full control over design customization. It allows me to create scalable and performance-optimized user interfaces without writing large custom CSS files.

React.js

React.js is my primary frontend library for building reusable and component-based user interfaces. I work with hooks, state management, props, and lifecycle methods to create fast and interactive web applications.

Node.js

Node.js allows me to build scalable and high-performance backend services. I use Node.js to handle server-side logic, manage APIs, and process asynchronous requests efficiently.

Express.js

Express.js is a lightweight backend framework that I use to build RESTful APIs. It helps in routing, middleware handling, authentication, and managing server-side workflows efficiently.

MongoDB

MongoDB is a NoSQL database that I use to store and manage application data. I work with collections, documents, schema design, and CRUD operations to build flexible and scalable database-driven applications.

GSAP & Web Animations

GSAP helps me create smooth and professional animations such as scroll-based effects, hover interactions, and page transitions. These animations enhance user experience and give a modern look to web applications.
